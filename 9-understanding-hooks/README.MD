# useState -

state ko manage karna .

# useEffect -

side effect handle karne ke liye (jaise API calls,DOM manipulation,event lisenter).

# useContext -

global state ko consume karne ke liye without props drilling.

# useReducer -

complex state management ke liye (Redux jesa chhota version).

# useRef -

mutable value hold karne ke liye jo render na karien , ya DOM access karne liye.

# useMemo & useCallback -

Optimization ke liye , Unnecerry re-renders avoid karne ke liye .
